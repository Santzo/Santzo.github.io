Technologies

Unity

Unity is the game engine that really got me into developing software. I use Unity almost every single day. Recently i've done research of Unitys new incoming Entity Component System, JOBS (for multi-threading) and UIElements.

C#

Alongside Unity, I obviously had to pick up and learn C#. I feel very comfortable working with C#, I definitely prefer static typed languages over dynamically typed ones. I have plenty of experience with C# and have also ventured a little bit into .NET and .NET Core technologies.

Javascript

Javascript is an obvious programming language to learn for anyone into web developing, so that was one of the biggest reasons I decided to learn it. I enjoy the syntax (since it resembles C# a lot) and also the immense flexibility of this language.

React / React Native

React (Native) was the first Javascript framework I learned. I'm really confident working in this enviroment and working with React is something I definitely want to do in the future.

Blender

While still a beginner in 3D modeling, I definitely have the skills to produce simple to moderately challenging 3D models. In the future I wish to be able to work a lot more with Blender.

Finished projects

Speedsouls Records

Developed with: React Native, Javascript

Platform: Android

Being a passionate speedrunner of games in the past, I decided to make this app for everyone that is into From Softwares souls series games (Dark Souls etc.). It fetches information from speedrun.com api. The purpose is to list all souls game related leaderboards. The application also allows searching users by their name.

Check this project out!
Infinirunner

Developed with: Unity, C#

Platform: Android

Infinirunner is the first finished game i've done. It's an infinite runner style game, where the goal is to survive for as long as possible while collecting points by killing incoming enemies and dodging obstacles. The game will always end with the death of the player. Has an online leaderboard that allows you to see how well you've done against other players.

Check this project out!
